# Linux Setup Assistant 🚀
*A universal script to automate Linux configuration across distros.*

Automate your Linux setup with this **interactive script**—no more manual `apt`, `dnf`, or `pacman` commands! Choose your distro, and let it handle:
- Gaming dependencies (Steam, Proton, Vulkan)
- Development tools (Python, Git, CMake)
- Multimedia (VLC, FFmpeg)
- Flatpak apps (Spotify, VS Code)

---

## 🔧 Features
✅ **Auto-detects distro** (Debian/Ubuntu, Arch, Fedora, OpenSUSE)
✅ **Interactive menu** (no CLI expertise needed)
✅ **Preconfigured profiles** (gaming, dev, multimedia)
✅ **System updates first** (avoids dependency conflicts)
